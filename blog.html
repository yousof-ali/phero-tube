1. Discuss the scope of var, let, and const.

Ans: The scope of a var variable is functional or global scope. The scope of a let variable is block scope. The scope of a const variable is block scope.

2. Tell us the use cases of null and undefined.

Ans: Null is basically an assignment value. The variable which has been assigned as null contains no value and is empty. Undefined is when we declare a variable but do not assign a value to it(define it), the variable becomes undefined

1. What do you mean by REST API?

Ans: A REST API (also known as RESTful API) is an application programming interface (API or web API) that conforms to the constraints of REST architectural style and allows for interaction with RESTful web services.
